¿Qué utilidades tiene Perifinder?

Perifinder es una aplicación web que ofrece varias herramientas. Permite comparar características entre varios componentes del mismo tipo y también crear un presupuesto (excluyendo la carcasa) para armar un ordenador.

¿Cómo se utiliza el Comparador?

Para utilizar el comparador, selecciona un componente en uno de los dos buscadores disponibles. La selección cambiará la estructura del buscador. El otro buscador te mostrará opciones disponibles según el componente seleccionado. La comparación se muestra como un modelo Versus con marcas favorables (verde) y desfavorables (rojo). Además, al final se realizará un cálculo de los componentes seleccionados y te mostrará el componente con las mejores características.

¿Cómo se utiliza el Budget Maker?

Para utilizar el Budget Maker, simplemente incluye los componentes que deseas añadir en la lista. Junto a cada componente en la lista, verás el precio. A medida que agregues componentes, verás el precio total justo debajo de la imagen.
