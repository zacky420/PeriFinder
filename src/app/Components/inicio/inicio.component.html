Qué Utilidades tiene Perifinder?

Perifinder es una aplicación web con varias herramientas en las cuales permite comparar características entre varios componentes del mismo tipo, como también crear un presupuesto (sin incluir la carcasa) de un ordenador en las cuales quieras montarte.
